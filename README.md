## Getting Started

### Prerequisites
- Node.js 18.0 or higher
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/bikingsinghrathor/bikingsinghrathor.github.io.git
cd bikingsinghrathor.github.io
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build for production
- `npm start` - Start the production server
- `npm run lint` - Run ESLint to check code quality

## Design Highlights

- **Animated Gradient Background**: A continuously shifting gradient background that creates a vibrant, energetic atmosphere
- **Interactive Profile Image**: Hover effects with smooth scaling and rotation
- **Smooth Transitions**: All elements feature smooth hover transitions for better user experience
- **Responsive Gallery**: Auto-fit grid layout that adapts to different screen sizes
- **Social Integration**: Direct links to social media profiles with animated hover effects
- **Mobile Optimized**: Fully responsive design that works on all device sizes

## Deployment

The site is optimized for deployment to Vercel, GitHub Pages, or any static hosting service.

### Deploy to Vercel
```bash
npm install -g vercel
vercel
```

### Build for Static Export
```bash
npm run build
```

The `out` directory will contain the static HTML files ready for deployment.
